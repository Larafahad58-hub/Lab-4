# Lab-4
In this assignment, we focus on applying data preprocessing techniques to a dataset using Python’s Pandas, Seaborn, Matplotlib, and Scikit-learn libraries. The goal is to explore and prepare the Titanic dataset for machine learning by performing various preprocessing tasks, including:
	1.	Handling Missing Values: We deal with missing data by either removing rows containing missing values or using imputation techniques to fill in the gaps with the mean or most frequent values.
	2.	Data Scaling and Standardization: The data is scaled using MinMaxScaler and StandardScaler to ensure that the features are on a similar scale, making them suitable for machine learning models that are sensitive to feature scaling.
	3.	Principal Component Analysis (PCA): PCA is applied to reduce the dimensionality of the dataset while retaining the most important features. This helps in visualizing high-dimensional data and can improve the performance of machine learning algorithms by reducing noise.
	4.	Visualization: We use heatmaps and scatter plots to visualize correlations between features and the results of PCA, providing valuable insights into the structure of the data.
